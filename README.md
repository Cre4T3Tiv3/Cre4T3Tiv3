#### [Unsloth LLaMA 3 Adapter (QLoRA)](https://github.com/Cre4T3Tiv3/unsloth-llama3-alpaca-lora)

Fine-tuned with [Unsloth](https://github.com/unslothai/unsloth), built for low-resource inference and agentic workflows.
